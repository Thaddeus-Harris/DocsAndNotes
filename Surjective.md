## Def
- f is surjective or is a surjection if for all $b \in B$ there is some $a \in A$ such that f(a) = b, i.e., the image off is all of B. Note that since a function always maps onto its range (by definition) it is necessary to specify the codomain B in order for the question of surjectivity to be meaningful.
